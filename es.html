Algunos proyectos paralelos

uNotifier

Bot simple de Telegram que te permite, a través de una simple API REST, enviar mensajes a cualquier usuario que haya generado y compartido un token. Esto te ayuda a implementar una librería de notificaciones a usuarios gratis y en unos pocos minutos, sacando ventaja de la infraestructura de esta aplicación.

El projecto consta de un programa Node.js dockerizado con una CI que lo despliega automáticamente en cada commit. Está alojado en una instancia VPS de la AWS usando nginx como un proxy inverso. Las imágenes de docker están almacenadas en la AWS ECR, y los datos en una base de datos PostgreSQL.

Versioncloud

Aplicación web para compartir archivos con un estricto control de versiones. Su infraestructura es muy parecida a uno de mis otros proyectos, Unotifier, pero además usa buckets AWS S3 para almacenar algunos archivos. El frontend server-side-rendering está escrito en Vue y también está dockerizado.

Game Jams

He participado como programador en diferentes Game Jams, eventos organizados en internet destinados a crear un pequeño videojuego en una cantidad limitada de tiempo, normalmente 3 días. Junto a diseñadores de juego e ilustradores, he programado estos juegos en Unity, Game Maker, e incluso en un pequeño motor de juegos que desarrollé yo mismo en JavaScript.
